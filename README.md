# Marketing Campaign Effectiveness Study

## Project Overview

This project analyzes a marketing campaign dataset to understand customer behavior and evaluate the effectiveness of marketing strategies. The goal is to identify patterns in customer demographics, purchasing behavior, and campaign responses to help businesses improve their marketing decisions.

The analysis includes exploratory data analysis (EDA), data visualization, and a machine learning model to predict whether a customer will accept a marketing campaign.

---

## Dataset

The dataset contains customer information related to marketing campaigns and purchasing behavior.

**Key Features in the Dataset**

* Customer demographics (Education, Marital Status, Income)
* Purchase behavior (Wine, Fruits, Meat, Fish, Sweets, Gold products)
* Purchase channels (Web, Store, Catalog)
* Campaign response indicators
* Customer engagement metrics (Recency, Web visits)

**Target Variable**

* `Response` – Indicates whether a customer accepted the marketing campaign.

## Project Workflow

1. **Data Loading**

   * Import the marketing dataset.

2. **Data Preprocessing**

   * Handle missing values.
   * Convert categorical data.
   * Prepare features for machine learning.

3. **Exploratory Data Analysis (EDA)**

   * Analyze customer demographics.
   * Study spending patterns.
   * Examine campaign response distribution.

4. **Data Visualization**

   * Campaign response distribution
   * Income vs Response analysis
   * Product spending comparison

5. **Machine Learning Model**

   * Feature encoding
   * Train-test split
   * Random Forest Classifier model

6. **Model Evaluation**

   * Accuracy score
   * Prediction analysis

---

## Key Insights

* Most customers did not accept the marketing campaign, indicating a low conversion rate.
* Higher income customers are more likely to respond positively to marketing offers.
* Wine and meat products generate the highest customer spending.
* Store purchases are higher than web and catalog purchases.
* Recently active customers are more likely to accept marketing campaigns.
